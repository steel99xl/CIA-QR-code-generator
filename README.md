# CIA-QR-code-generator
Hosts a simple web server and generates QR codes to scan with FBI 3ds homebrew

# How to use
Run QRMaker.py in the same folder as your .cia files

```py ./QRMaker.py```

# Install
dependancies

```pip3 install -r requirements.txt```
